# Критерии оценки "стало лучше"

База сравнения: `gpt-4o-mini` без ФТ vs `ft:gpt-4o-mini:*` после ФТ.
Замер: 10 примеров из `data/eval.jsonl`, temperature=0.

## 1. Accuracy (главная метрика)

**Определение**: доля случаев когда модель вернула ровно ожидаемый label.

- **Baseline целевой потолок**: ~70% (ожидаемо, gpt-4o-mini путает understand/describe и search/modify)
- **ФТ целевой порог**: ≥ baseline + 10 п.п. (т.е. если baseline = 70%, ФТ ≥ 80%)
- **Формула**: `sum(actual == expected) / len(eval)`

Accuracy per class:
- Считаем отдельно для каждого из 4 классов (по 2-3 примера на класс в eval)
- Требование: **ни один класс не падает ниже 50%** после ФТ

## 2. Формат ответа (strict match)

**Определение**: модель возвращает ровно один из 4 ярлыков, без лишних символов.

- Baseline типично нарушает: ответы вида `"search"`, `"Это search"`, `search.`, `search\n`, `"search" (объяснение: ...)`
- **ФТ целевой порог**: 100% ответов соответствуют регулярке `^(search|understand|describe|modify)$`
- Проверка применяется **после** нормализации (strip, lowercase)

## 3. Стабильность

**Определение**: temperature=0 → один и тот же вход даёт один и тот же выход на двух прогонах.

- **Требование**: 10/10 совпадений между двумя независимыми прогонами
- Если не совпадает — проблема в API (редкий race), не в модели

## 4. Покрытие классов

**Определение**: модель не сваливается в один доминантный класс.

- Baseline часто предсказывает `modify` на всё что содержит глагол действия
- **Требование**: в предсказаниях ФТ-модели на eval встречаются **все 4 класса** (или минимум 3 если в eval отсутствует один)

## 5. Устойчивость к стилю

**Определение**: модель одинаково справляется с разными стилями промптов.

Подмножества eval по стилю (проверяем вручную на финальном прогоне):
- **Короткие** (< 30 симв): `"где код девайса?"`, `"у нас macos таргет какой?"` → ожидаем search
- **С кодом**: содержат фрагмент кода или название функции → класс определяется контекстом
- **Грубые/колкие**: реальные промпты с разговорным тоном → классификация не должна страдать

## Сводка "зелёный флаг" (все должны выполниться одновременно)

1. ✅ accuracy ФТ ≥ baseline + 10 п.п.
2. ✅ ни один класс не ниже 50% accuracy
3. ✅ 100% ответов соответствуют формату
4. ✅ стабильность 10/10
5. ✅ в предсказаниях присутствуют минимум 3 класса из 4
